# Python CSV CRM Cleaner

A Python script to clean and prepare messy CSV files before CRM import.

## Features

- Reads large CSV files with inconsistent formatting  
- Deduplicates contacts based on name and email  
- Keeps only the most recent record when duplicates exist  
- Validates required fields  
- Outputs a clean CSV file ready for CRM import  

## Use Case

Helps prevent data corruption and manual cleanup during CRM imports.

## Tech

- Python  
- Pandas
